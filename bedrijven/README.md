# Bedrijven

Een nieuw project voor het maken van websites.

## Bestanden

- `index.html` — de startpagina van de website
- `css/style.css` — de opmaak (kleuren, lettertype, layout)

## Hoe bekijk je de website?

Open het bestand `index.html` in je internetbrowser (bijvoorbeeld Chrome of Edge).

## Wat nu?

Vertel Claude wat je wilt toevoegen of veranderen, bijvoorbeeld:
- een lijst met bedrijven
- foto's of een logo
- een menu of contactpagina
